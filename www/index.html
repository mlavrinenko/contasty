contasty

Strips executable lines from your code to prepare tasty context for your agent.

GitHub crates.io Docs

What it does

contasty walks a directory (respecting .gitignore), parses each supported source file with ast-grep, and elides function bodies, constant values, and long strings — keeping declarations, signatures, and types intact. The result is a single Markdown (or JSON) bundle, ready to paste into an LLM context window.

Install

cargo install contasty

Or download a pre-built binary from the latest release.

Highlights

  • Built-in Rust and PHP, driven by YAML ast-grep rule files.
  • Add a language with a dynamic .so grammar — no rebuild.
  • Extend or override any language's rules from contasty.toml.
  • Markdown or JSON output; drop tests, comments, and imports on demand.
